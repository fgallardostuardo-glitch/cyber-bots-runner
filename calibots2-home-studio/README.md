# Calibots 2 Home Studio

Responsive English study app for a 5-year-old child, inspired by the oral and visual style of Calibots 2 home practice.

## What changed
- Numbers now go from **1 to 10**.
- The app keeps **pre-reader** and **reader support** modes.
- It is packaged as a static web app that works well on desktop and Android browsers.

## Files
- `index.html` → full app in one file
- `assets/favicon.svg` → app icon

## Notes
This hosted version is a lightweight web build prepared for easy use from a public repository folder.
